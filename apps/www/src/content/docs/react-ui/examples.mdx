These examples show complete composition patterns rather than isolated primitives. Use them as
starting points for app-owned chat, completion, tool-call, and review experiences.

## Recipes

- [Full chat surface](/docs/react-ui/examples/chat-surface): thread layout, suggestions, default
  message rendering, composer, and scroll controls.
- [Message rendering](/docs/react-ui/examples/message-rendering): default renderer, Markdown,
  custom code blocks, attachments, tool cards, and actions.
- [Styling recipe](/docs/react-ui/examples/styling): complete Vanilla CSS and Tailwind chat
  styling.
- [Composer attachments](/docs/react-ui/examples/composer-attachments): file picker, direct file
  input, dropzone, controlled attachments, and custom attachment rows.
- [Tool and human input](/docs/react-ui/examples/tool-human-input): tool cards, pending/settled
  filtering, approvals, and questions.
- [Completion panel](/docs/react-ui/examples/completion): prompt-to-text completion surface with
  custom output rendering.
