## State selectors

Use `data-state`, `data-role`, and `data-anvia-*` selectors for state-specific styling.

```css
[data-anvia-composer][data-state="streaming"] [data-anvia-submit] {
  display: none;
}

[data-anvia-scroll-to-bottom][data-state="bottom"] {
  visibility: hidden;
}

[data-anvia-tool][data-state="error"] {
  border-color: #fb7185;
}
```

Keep the package stylesheet imported for behavior, then put product layout and visual rules in your
app CSS or Tailwind classes.
